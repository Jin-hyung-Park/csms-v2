## 🔍 추가 확인 권장 사항

다음 항목들은 선택적으로 확인할 수 있습니다:

1. **보호된 페이지 접근 제어**
   - 로그인 없이 `/employee/dashboard` 접근 시 리다이렉트 확인
   - 로그인 후 보호된 페이지 접근 확인

2. **토큰 관리**
   - localStorage에 토큰 저장 확인 (개발자 도구)
   - API 요청 시 자동 토큰 추가 확인 (Network 탭)

3. **WorkSchedule API 동작**
   - 로그인 후 근무일정 조회/생성/수정/삭제 동작 확인

4. **에러 처리**
   - 잘못된 이메일/비밀번호 로그인 시 에러 메시지 확인
   - 중복 이메일 회원가입 시 에러 메시지 확인
