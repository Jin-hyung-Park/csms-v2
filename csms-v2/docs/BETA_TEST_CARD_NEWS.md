## 회원가입 

**✏️ 회원가입**

화면에서 **「회원가입」** 을 누르고 **역할**에서 **「근로자(직원)」** 를 선택하세요.

**🔑 매장코드 입력**

매장코드는 **CU002**를 입력한 뒤  **「검증」** 버튼을 눌러요.  
✅ 매장 이름이 뜨면 정상입니다.  


**📝 정보 입력 후 가입**

**이름 · 이메일 · 비밀번호 · 전화번호**  를 입력하고 **「가입하기」** 를 누르세요.

가입 요청이 완료되면 점주 **승인**을 기다리면 됩니다.

⏳ 점주가 승인하면 하단에 **홈 · 근무표 · 급여 · 내정보** 메뉴가 열립니다.  
🔐 승인해도 메뉴가 안보인다면, 다시 로그인해주세요.



## 근무이력 입력 방법

**📅 근무이력 입력**

앱 하단 **「근무표」** → **「근무일정 등록」**

- **날짜** 
- **시작 시간 · 종료 시간**  
입력 후 **저장**하면 점주에게 승인 요청이 갑니다.
