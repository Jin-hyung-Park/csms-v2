# 삼성메가 점포 근로자용 CSMS v2 사용 가이드

> **편의점 근무 관리 앱** — 근무일정 등록, 급여 확인, 알림까지 한곳에서  
> 배포일 기준으로 앱 주소는 점주/관리자에게 안내받으세요.

---

## 1. 앱 접속

- **접속 주소**: [배포 후 점주가 안내하는 주소로 교체해 주세요]
- 휴대폰 브라우저(Chrome, Safari 등)에서 주소 입력 후 **홈 화면에 추가** 해 두면 앱처럼 사용할 수 있습니다.

---

## 2. 최초 가입 (신규 근로자)

1. 접속 후 **「회원가입」** 을 누릅니다.
2. **역할**에서 **「근로자(직원)」** 를 선택합니다.
3. **매장코드(5자리)** 를 입력합니다.
   - 삼성메가 점포의 매장코드는 **점주(店長)에게 받습니다**.
   - 5자리 코드 입력 후 **「검증」** 버튼을 누르면, 해당 매장 이름이 나오면 정상입니다.
4. **이름, 이메일, 비밀번호, 전화번호**를 입력합니다.
5. **「가입하기」** 를 누르면 **가입 요청**이 완료됩니다.
6. 점주가 승인하기 전까지는 **「가입 승인 대기 중」** 안내만 보입니다. 점주 승인 후 로그인하면 모든 메뉴를 사용할 수 있습니다.

---

## 3. 로그인

- **이메일**과 **비밀번호**로 로그인합니다.
- 가입 후 **점주 승인 전**에는 대시보드에서 「가입 승인 대기 중」 문구만 보이고, 승인 후에는 하단 메뉴(홈, 근무표, 급여, 내정보)가 보입니다.

---

## 4. 메뉴 사용법

### 4.1 홈 (대시보드)

- **근무 점포·계약 정보**: 소속 점포, 근무 요일/시간, 시급 등을 한눈에 볼 수 있습니다.
- **이번 주 근무 현황**: 이번 주 총 근무시간, 근무일수, 예상 급여, 승인 대기/확정 건수입니다. 카드를 누르면 **근무표** 화면으로 이동합니다.
- **지난달 급여 요약**: 지난달 근무시간·예상 급여 등 간단 요약입니다. 카드를 누르면 **급여** 화면으로 이동합니다.

### 4.2 근무표

- **등록**: 「근무일정 등록」에서 **날짜, 시작/종료 시간**을 입력한 뒤 **저장**하면 점주에게 승인 요청이 갑니다.
- **조회**: 월을 선택하면 주별로 등록한 근무일정을 볼 수 있습니다. **승인/대기/거절** 상태가 표시됩니다.
- **수정/삭제**: 아직 승인되지 않은 일정은 수정·삭제할 수 있습니다. 승인된 일정은 점주가 수정할 수 있습니다.

### 4.3 급여

- **월별 급여 요약**: 월을 선택하면 해당 달의 **기본급, 주휴수당, 복지포인트, 예상 실수령액**을 볼 수 있습니다. 점주가 산정한 급여가 있으면 그 내용이 그대로 표시됩니다.
- **월별 상세 보기**: 주차별 근무시간, 기본급, 주휴수당, 세금, 실수령액을 자세히 볼 수 있습니다.
- **급여 확인 (이상 없음)**: 점주가 급여를 산정한 경우, 내용을 확인한 뒤 **「이상 없음 · 확인」** 버튼을 누르면 확인 완료로 저장됩니다. (나중에 분쟁 방지를 위해 보관됩니다.)
- **이상 시 피드백**: 급여 내용에 문제가 있으면 **「이상 시 피드백 보내기」** 로 점주에게 메시지를 보낼 수 있습니다. 보낸 내용도 보관됩니다.
- **Excel 다운로드**: 급여가 산정된 달은 **「📥 Excel」** 버튼으로 급여 명세서를 받을 수 있습니다.

### 4.4 내정보

- **이름, 이메일, 전화번호, 소속 점포**와 **근로 계약 정보**(근무 요일, 근무 시간, 시급 등)를 확인할 수 있습니다.

### 4.5 알림

- **알림** 버튼(🔔)을 누르면 **받은 알림 목록**을 볼 수 있습니다.
- 근무일정 승인/거절, 급여 확정, 점주가 보낸 메시지 등이 여기 표시됩니다.
- 알림을 열면 **읽음** 처리되고, 점주는 읽었는지 확인할 수 있습니다.

---

## 5. 자주 묻는 질문

**Q. 매장코드를 모르겠어요.**  
A. 삼성메가 점포 점주(店長)에게 5자리 매장코드를 요청하세요.

**Q. 가입했는데 메뉴가 안 보여요.**  
A. 점주가 가입을 승인해야 합니다. 「가입 승인 대기 중」이면 점주에게 승인 요청을 해 주세요.

**Q. 급여가 점주가 말한 금액과 달라요.**  
A. 급여 상세 화면에서 **「이상 시 피드백 보내기」** 로 점주에게 내용을 보내 주세요. 점주가 확인 후 조정할 수 있습니다.

**Q. 앱 주소를 잊어버렸어요.**  
A. 점주 또는 관리자에게 배포된 앱 주소를 다시 안내받으세요.

## 6. 요약

| 하고 싶은 일         | 하는 곳        |
|----------------------|----------------|
| 처음 가입            | 로그인 화면 → 회원가입 (매장코드 필요) |
| 이번 주·이번 달 근무 보기 | 홈 → 근무표   |
| 근무일정 넣기/수정   | 근무표         |
| 급여 보기·확인·피드백 | 급여           |
| 계약 정보 보기       | 내정보         |
| 일정 승인·메시지 확인 | 알림(🔔)       |
