## 1. 근로소득 세금이란?

### 1.1 개념

- **근로소득**: 임금·봉급·상여 등 **근로의 대가**로 받는 소득.
- **원천징수**: 편의점 등 사업장이 직원 급여를 지급할 때, **국가가 정한 세율**에 따라 소득세·지방소득세를 미리 떼어 납부하는 제도.
- **간이세액표**: 국세청이 배포하는 표로, **과세소득 구간별**로 원천징수할 **소득세액**을 적어 둔 것.  
  → 월 급여(과세소득)만 알면, **해당 구간의 세액**을 표에서 찾아 적용하면 됨.

### 1.2 사업자소득과의 차이

| 구분 | 근로소득 (labor-income) | 사업자소득 (business-income) |
|------|-------------------------|-----------------------------|
| 대상 | 임금·상여 등 근로 대가 | 사업 활동으로 인한 소득 |
| 세율 | **구간별 누진세율** (0% ~ 45%) | **비례세** 3.3% (소득세+지방세) |
| 특징 | 소득이 늘수록 세율이 단계적으로 올라감 | 소득 크기와 관계없이 3.3% |

편의점 **파트타이머**는 보통:
- **근로소득**으로 신고하면 → 간이세액표(구간별 누진) 적용.
- **사업자소득**으로 신고하면 → 3.3% 단일 세율 적용.

## 2. CSMS v2에서의 “근로소득 세금 계산” 기능

### 2.1 하는 일

- 점주가 직원의 **세금 유형**을 `labor-income`(근로소득)으로 설정한 경우,
- 해당 직원의 **월 급여(기본급 + 주휴수당)**를 기준으로:
  - **소득세**와 **지방소득세**(소득세의 10%)를 계산하고,
  - **실수령액** = 총 지급액 − 소득세 − 지방소득세 로 산출합니다.
- 이 값은 **MonthlySalary**의 `taxInfo`(incomeTax, localTax, totalTax, netPay)에 저장되고,  
  급여 명세·엑셀·화면에 그대로 사용됩니다.

### 2.2 현재 구현 상태 (소득세법 제55조 기준 보강 완료)

- **파일**: `server/src/utils/taxCalculator.js`
- **함수**: `calculateLaborIncomeTax(monthlyGrossPay)`
- **로직**:  
  **2024년 소득세법 제55조** 과세소득 구간별 세율을 **월 급여**에 적용. 구간별 누적세액 + 해당 구간 초과금액 × 세율로 소득세 산정 후, 10원 미만 절사. 지방소득세 = 소득세 × 10% (10원 절사).

| 월 과세소득 구간 (원) | 세율 | 해당 구간 누적세액(하한 기준) |
|------------------------|------|------------------------------|
| 0 ~ 140만 | 6% | 0 |
| 140만 초과 ~ 500만 | 15% | 84,000 |
| 500만 초과 ~ 880만 | 24% | 624,000 |
| 880만 초과 ~ 1,500만 | 35% | 1,536,000 |
| 1,500만 초과 ~ 3,000만 | 38% | 3,706,000 |
| 3,000만 초과 ~ 5,000만 | 40% | 9,406,000 |
| 5,000만 초과 ~ 8,800만 | 42% | 17,406,000 |
| 8,800만 초과 ~ 1억 | 45% | 38,406,000 |
| 1억 초과 | 45% | 43,806,000 |

- **지방소득세**: 소득세 × 10% (10원 단위 절사)
- **연동**:  
  `calculateMonthlyTax(taxType, monthlyGrossPay)` 에서 `taxType === 'labor-income'` 일 때 위 함수를 호출하며,  
  `monthlySalary.route.js` 급여 산정 시 `employee.taxType`에 따라 이 계산이 적용됩니다.

즉, **“근로소득 세금 계산”**은 **과세소득 구간별 세율**로 산정되며, 점주가 세금 유형을 **근로소득**으로 두면 위 표 기준으로 세금이 계산·저장·표시됩니다.

## 3. 간이세액표와의 관계

- **간이세액표**: 국세청이 월급여액·공제대상가족 수별로 **세액을 미리 계산해 둔 표**. 원천징수 시 해당 칸의 세액을 그대로 사용.
- **현재 구현**: **소득세법 제55조**의 과세소득 구간별 **세율**을 월 급여에 적용해 같은 구조로 세액을 산정. 구간·세율이 법령과 일치하므로, 간이세액표와 완전히 동일한 수치는 아니더라도 **과세소득 구간별 세율에 따른 산정**이 이루어짐.
- **선택적 보강**: 공제대상가족 수를 입력받아 간이세액표의 가족 수별 세액을 그대로 참조하는 방식으로 확장할 수 있음.

## 4. 선택적 보강 시 참고 사항

- **국세청 간이세액표**:  
  연도별·공제대상가족 수별로 표가 나뉘므로,  
  정확 반영 시에는 **연도**, **공제대상가족 수** 등을 입력받아 해당 표를 참조하는 방식이 필요합니다.
- **현재 코드**:  
  공제대상가족 0명, 2024년 근사 세율로 고정되어 있으므로,  
  필요하면 “간이세액표 정확 반영” 시 **표 데이터(JSON/상수)** 와 **가족 수 등 파라미터**를 확장하면 됩니다.
