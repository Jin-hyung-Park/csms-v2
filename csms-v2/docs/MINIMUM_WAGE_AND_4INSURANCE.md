# 최저시급 및 4대 보험 기능 명세

> 2026년 적용: 최저시급 10,320원, 4대 보험 산정식

---

## 1. 최저시급 설정

### 1.1 배경
- 2026년 최저시급이 10,030원에서 **10,320원**으로 인상됨.
- 최저시급은 매년 변경되므로 **점주가 최저시급을 설정**할 수 있는 기능이 필요함.

### 1.2 기능 요구사항
- 점주는 **점포별**로 적용 최저시급(원/시간)을 설정할 수 있음.
- 점포에 설정된 최저시급은 해당 점포 소속 근로자의 **시급 기본값**으로 사용됨.
- 시스템 기본값: **10,320원** (2026년 기준).

### 1.3 구현 요약
- **Store** 모델: `minimumWage` 필드 (Number, 기본값 10,320).
- 점포 등록/수정 시 `minimumWage` 입력 가능.
- 근로자 시급 미설정 시 해당 점포의 `minimumWage` 또는 10,320원 사용.

---

## 2. 근로자 세금 유형 – 4대 보험 대상

### 2.1 4대 보험 구성
- **건강보험** (건강보험료)
- **국민연금** (국민연금 보험료)
- **고용보험** (고용보험료)
- **산재보험** (산재보험료) – 본 산정식에서는 고용보험과 함께 고용보험료로 처리

추가로 **소득세·지방소득세**를 동일한 “대상 급여” 기준으로 산정함.

### 2.2 산정식 (대상 급여 = 월 총 급여: 기본급 + 주휴수당 등)

| 항목 | 산식 | 절사 규칙 |
|------|------|-----------|
| 국민연금 | 대상 급여 × 4.5% | 1,000원 미만 절사 |
| 건강보험 | 대상 급여 × 3.545% | 10원 미만 절사 |
| 장기요양보험 | 산정된 건강보험료 × 12.95% | 10원 미만 절사 |
| 고용보험 | 대상 급여 × 0.9% | 10원 미만 절사 |
| 소득세 | 대상 급여 × 1.53% | 10원 미만 절사 |
| 지방소득세 | 산정된 소득세 × 10% | 10원 미만 절사 |

- **절사**: 해당 단위 미만은 버림 (국민연금 1,000원 단위, 나머지 10원 단위).
- **실수령액** = 대상 급여 − (국민연금 + 건강보험 + 장기요양 + 고용보험 + 소득세 + 지방소득세).

### 2.3 세금 유형(taxType) 값
- 기존: `none`, `under-15-hours`, `business-income`, `labor-income`
- 추가: **`four-insurance`** (4대 보험 대상)
- 근로자 직원 정보(시급·세금 유형) 설정 시 “4대 보험 대상” 선택 가능.
