# 복지포인트 산출 및 급여 엑셀 포맷

> 전주 실 근로시간 기반 복지포인트, 매장별·근로자별 급여 엑셀 포맷 명세

---

## 1. 복지포인트 산출

### 1.1 규칙
- **전주 실 근로시간**을 기반으로 **차주**에 복지포인트를 제공합니다.
- **산식**: `복지포인트 = trunc(실 근로시간 / 4, 0) × 1,700원`
  - `trunc(·, 0)`: 소수 이하 버림 (정수만 사용)
  - 예: 10시간 근무 → floor(10/4)*1700 = 2*1700 = **3,400원**

### 1.2 적용
- 월별 급여의 **주차별 실 근로시간**에 위 산식을 적용하여, 해당 월에 적립되는 복지포인트 합계를 산출합니다.
- **점주/근로자** 급여 메뉴(급여 목록·상세)에서 복지포인트를 확인할 수 있습니다.

### 1.3 저장
- **MonthlySalary** 모델에 `totalWelfarePoints`(Number) 필드로 월 합계를 저장합니다.
- 산정 시 각 주차 `workHours`에 대해 `floor(workHours/4)*1700`을 합산하여 저장합니다.

---

## 2. 급여 메뉴 엑셀 다운로드 포맷

### 2.1 구성
- **매장별·근로자별 급여내역** 한 시트로 제공합니다.

### 2.2 컬럼 정의

| 컬럼명 | 설명 | 비고 |
|--------|------|------|
| 매장(점포명) | 소속 점포 이름 | |
| 이름 | 근로자 이름 | |
| 주민번호 | 주민등록번호 | User.ssn (선택) |
| 입사일 | 입사일 | User.hiredAt (선택) |
| 근로계약상 주단위 근로시간 | 계약상 주당 근로시간(시간) | User.workSchedule 합산 |
| 세금유형 | taxType 라벨 | none/under-15-hours/business-income/labor-income/four-insurance |
| 해당월 근무시간 | 해당 월 실 근로시간(시간) | totalWorkHours |
| 총급여 | 기본급+주휴수당 등 총 지급액 | totalGrossPay |
| 실지급액 | 세금·4대보험 공제 후 실수령액 | taxInfo.netPay |
| 4대보험료 | 국민연금+건강+장기요양+고용보험 합계 | taxInfo 4항목 합계 |

### 2.3 점주/근로자
- **점주**: `GET /api/owner/export/payroll?year=&month=&storeId=` 로 위 포맷 엑셀 다운로드.
- **근로자**: `GET /api/employee/export/payroll/:year/:month` 로 본인 급여 명세서 다운로드(상세 형식 유지).
